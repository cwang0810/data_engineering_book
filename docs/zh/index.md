# 详细大纲内容

## 第一部分：总论与基础设施

> **目标：** 建立 Data-Centric AI 的系统认知，完成高性能数据处理环境的搭建与成本治理。

### 第1章：大模型时代的数据变革

- 1.1 **Scaling Laws 的启示：** 数据质量 > 数量，从"大数据"到"高质量数据"的范式转移。
- 1.2 **LLM 数据全生命周期：** 预训练(Pre-train) → 微调(SFT) → 强化学习(RLHF) → 检索增强(RAG)。
- 1.3 **挑战与机遇：** 异构多模态、版权合规与算力成本的博弈。

### 第2章：LLM 数据生命周期与质量评估框架

- 2.1 **数据全生命周期视角：** 采集 → 清洗 → 标注 → 迭代的闭环体系。
- 2.2 **数据质量多维评估：** 准确性、多样性、覆盖率与合规性的量化框架。
- 2.3 **质量驱动的迭代机制：** 如何用评估指标指导数据工程决策。

### 第3章：AI 原生数据栈与成本治理

- 3.1 **AI 原生数据栈 (AI-Native Data Stack)：**
  - 存储：对象存储 (S3/MinIO) vs 数据湖 (Iceberg/Hudi)。
  - 计算：Spark vs **Ray Data** vs **Dask** 三大分布式框架对比。
  - 向量数据库：Milvus / Qdrant / Weaviate / Pinecone 选型与 QPS vs Recall 权衡。
- 3.2 **数据格式与 I/O 优化：**
  - Parquet vs JSONL vs WebDataset（多模态场景）。
  - 压缩算法与读取性能优化，GPU 训练 I/O 瓶颈优化策略。
- 3.3 **成本治理：** 计算、存储、标注的全链路成本核算与优化策略。

## 第二部分：文本预训练数据工程

> **目标：** 处理海量无结构文本，构建模型的语言认知基座。

### 第4章：数据源、采集与版权

- 4.1 **开源数据集解构：** Common Crawl、C4、RefinedWeb、The Pile、DCLM 深度剖析。
- 4.2 **高性能爬虫系统：** `Trafilatura` 解析库应用与分布式爬虫架构设计。
- 4.3 **特种数据获取：** 代码 (GitHub)、论文 (ArXiv/S2ORC)、书籍数据的提取策略。
- 4.4 **版权合规：** 开源协议识别、Robots.txt 遵守与数据授权管理。

### 第5章：清洗、去重与去污染

- 5.1 **启发式过滤规则：** 语言识别 (FastText)、困惑度 (Perplexity) 过滤、长度与标点分布。
- 5.2 **大规模去重（精确 vs 模糊）：**
  - **精确去重：** 哈希方法快速移除完全相同文档。
  - **模糊去重 (Fuzzy Deduplication)：** MinHash LSH 算法原理与分布式实现。
- 5.3 **隐私清洗 (PII Removal)：** 使用 Presidio 识别并掩盖 Email、IP、电话、地址。
- 5.4 **基准测试集去污染：** 确保训练数据不包含 GSM8K、MMLU 等测试集原题。

### 第6章：分词、序列化与高效加载

- 6.1 **分词器原理：** BPE、WordPiece、Unigram 及 Byte-Level BPE 深入解析。
- 6.2 **高效词表构建：** 领域特定词表扩充与中文词表扩充工程实践。
- 6.3 **数据混合 (Data Mixing)：** 动态采样策略与 Curriculum Learning 数据排布。
- 6.4 **高性能 DataLoader：** 多进程预取、内存映射与 GPU 直传优化。

### 第7章：数据评估、质量闭环与运营迭代

- 7.1 **基于模型的质量评分：** 使用 fastText/BERT 进行"教科书级质量"打分。
- 7.2 **数据配方 (Data Recipe)：** 多来源混合比例的实验与优化。
- 7.3 **质量飞轮：** 评估结果回写驱动的数据迭代闭环机制。

## 第三部分：多模态数据工程

> **目标：** 处理图像、视频与音频，支持 GPT-4V/Sora 类模型的训练。

### 第8章：图文对数据工程

- 8.1 **数据范式：** 图文对 (LAION-5B) vs 交错文档 (OBELICS/MMC4)。
- 8.2 **图像获取与预处理：** `img2dataset` 高并发下载、GPU 加速解码 (NVIDIA DALI)。
- 8.3 **多模态清洗流水线：**
  - **美学评分 (Aesthetics)：** 使用 CLIP-Score 筛选高美感图片。
  - **图文对齐过滤：** 剔除描述与图片不符的样本。
  - **安全性检测：** NSFW 与水印识别。

### 第9章：重标注与文档理解

- 9.1 **Alt-text 的局限性：** 为什么原始网页描述不可用？
- 9.2 **合成描述工厂：** 利用 BLIP-2 / LLaVA / CogVLM 重新生成详细 Caption，Prompt 策略控制颗粒度。
- 9.3 **OCR 增强与文档理解：** 提取图中文字并融合进描述，支持文档图像理解任务。

### 第10章：视频与音频数据工程

- 10.1 **视频处理流水线：** 场景切分 (Scene Detection) 与关键帧提取策略。
- 10.2 **视频 Tokenization：** 视频压缩与离散化表示。
- 10.3 **音频对齐：** 使用 Whisper 进行大规模 ASR 及 Force Alignment（时间戳对齐）。

### 第11章：跨模态对齐与融合

- 11.1 **对齐目标：** 视觉-语言对齐的技术路径与评估标准。
- 11.2 **跨模态数据配比策略：** 图文/视频/音频的混合比例实验。
- 11.3 **多模态数据质量评估：** CLIP Score、语义一致性与对话合理性的自动化评估框架。

## 第四部分：指令微调与偏好数据

> **目标：** 让模型听懂指令并学会人类偏好，构建高质量对齐数据。

### 第12章：SFT 数据设计与指令体系

- 12.1 **Prompt Engineering 为数据生产服务：** 编写高鲁棒性的 System Prompt。
- 12.2 **自动化构造方法：** Self-Instruct 利用强模型生成指令；Evol-Instruct 指令复杂度进化策略。
- 12.3 **思维链 (CoT) 数据：** 构造 Step-by-Step 的推理样本。
- 12.4 **指令多样性与覆盖率：** 任务类型分布平衡与难度梯度设计。

### 第13章：偏好数据与奖励信号

- 13.1 **偏好数据格式：** Chosen vs Rejected 样本对构建与 DPO 数据标准。
- 13.2 **RLAIF (AI Feedback)：** 使用 LLM 代替人类进行偏好打分，Constitutional AI 方法论。
- 13.3 **奖励模型训练数据：** RM 数据采集策略与质量控制。

### 第14章：标注平台、QA 体系与数据运营

- 14.1 **标注平台选型：** Label Studio、Scale AI、Labelbox 对比与企业自建方案。
- 14.2 **标注一致性 (IAA)：** Cohen's Kappa、Fleiss' Kappa 的计算与质量把控。
- 14.3 **众包管理：** 标注员筛选、培训与质检抽样机制。
- 14.4 **数据运营机制：** 需求优先级管理、交付 SLA 与问题闭环流程。

## 第五部分：合成数据工程

> **目标：** 突破人类数据瓶颈，利用模型生成高质量训练数据。

### 第15章：合成数据工厂：从种子到验证

- 15.1 **教科书级数据 (Textbooks Are All You Need)：** 合成高质量领域知识的方法论。
- 15.2 **代码与数学合成：** PoT (Program of Thought) 生成代码并执行，以执行结果验证数据正确性。
- 15.3 **多模态指令合成：** 利用 GPT-4o 构造基于图像的复杂推理问答。
- 15.4 **合成数据的验证机制：** 自动化验证流水线与人工抽检策略。

### 第16章：知识蒸馏与模型协作

- 16.1 **蒸馏数据范式：** 从强模型（Teacher）向弱模型（Student）迁移能力的数据策略。
- 16.2 **模型协作生成：** 多模型投票、最佳-of-N 与 Critic 模型过滤。
- 16.3 **自我改进 (Self-improvement)：** 模型生成 → 验证 → 过滤 → 再训练的闭环。

### 第17章：合成数据质量控制与模型坍缩

- 17.1 **合成数据质量检测：** 多样性分析、分布偏移检测与幻觉率评估。
- 17.2 **模型坍缩 (Model Collapse)：** 成因分析与防止过拟合合成数据的工程策略。
- 17.3 **真实数据与合成数据的配比：** 混合训练的最优比例实验。

## 第七部分：应用级数据工程

> **目标：** 面向企业落地，解决外部知识库的解析、检索与反馈闭环。

### 第21章：RAG 数据流水线

- 21.1 **文档深度解析：** 复杂 PDF 处理（表格还原、多栏识别）、`Unstructured`/`LlamaParse` 实战。
- 21.2 **切片策略 (Chunking)：** 语义切片、递归切片与父子索引 (Parent-Child Indexing)。

### 第22章：多模态 RAG 与视觉检索

- 22.1 **跨模态检索：** 使用 CLIP/SigLIP 实现"以文搜图"与"以图搜文"。

### 第23章：在线反馈闭环与知识更新

- 23.1 **用户反馈采集：** 显式反馈（点赞/踩）与隐式反馈（停留时长、复制行为）的数据化。

## 第十部分：项目实战

> **目标：** 通过 10 个端到端项目，串联全书技术点，提供可运行的代码库。

### 项目一：基于 Ray 构建分布式 Mini-C4 数据流水线

- **场景：** 从 Common Crawl 原始数据 (WARC) 到高质量 Parquet 数据。
- **核心技术：** Trafilatura 解析、Ray 分布式 MinHash 去重、KenLM 质量过滤。
- **输出：** 清洗后的纯文本语料库与处理 Pipeline。

### 项目二：垂直领域专家 SFT（法律）

- **场景：** 基于非结构化 PDF 文档构建行业专家微调数据。
- **核心技术：** Self-Instruct 构造指令、CoT 推理增强、数据多样性平衡。
- **输出：** `domain_expert.jsonl` 指令微调集。

### 项目三：LLaVA 多模态指令数据工厂

- **场景：** 训练一个能看懂图片的多模态模型。
- **核心技术：** GPT-4o API 生成多轮图文对话、Bounding Box 数据对齐、多图交错格式处理。
- **输出：** 包含视觉指令的图文数据集。

### 项目四：合成数学与代码教材工厂

- **场景：** 提升小模型的逻辑推理能力。
- **核心技术：** Evol-Instruct 进化策略、Python 代码执行沙箱 (Sandbox) 验证、PoT 数据格式化。
- **输出：** 经过验证的高质量合成推理数据集。

### 项目五：多模态 RAG 企业财报助手

- **场景：** 检索并回答包含复杂图表的年度财报问题。
- **核心技术：** PDF 表格与图表解析、多路召回（混合检索）、ColPali 视觉检索应用。
- **输出：** 一个支持图表问答的 RAG 知识库系统。

### 项目六：CoT 推理数据集构建与 PRM 训练

- **场景：** 构建数学推理过程数据并训练过程奖励模型。
- **核心技术：** 自动化 CoT 生成、步骤级标注、PRM 训练数据流水线。
- **输出：** 推理过程数据集与 PRM 模型。

### 项目七：Agent Tool-Use 数据工厂

- **场景：** 构建多工具调用的 Agent 训练数据。
- **核心技术：** 工具调用链自动采集、轨迹标注、多步任务数据验证。
- **输出：** Agent 工具调用训练数据集。

### 项目八：企业级 DataOps 平台搭建

- **场景：** 从数据项目到组织级治理能力的平台化落地。
- **核心技术：** Airflow 调度、DVC 版本管理、Great Expectations 质量监控。
- **输出：** 企业级数据运营体系与平台架构方案。

### 项目九：隐私保护数据流水线

- **场景：** 构建满足合规要求的训练数据处理体系。
- **核心技术：** 联邦学习框架、差分隐私 DP-SGD、PII 自动脱敏流水线。
- **输出：** 合规训练数据流水线与隐私审计报告。

### 项目十：端到端 LLM 数据飞轮

- **场景：** 在线反馈驱动的持续数据迭代与模型提升。
- **核心技术：** 用户反馈采集、自动标注、增量训练与效果评估闭环。
- **输出：** 端到端 LLM 数据飞轮系统。
