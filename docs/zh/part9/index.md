# 第九篇：数据资产、数据产品与数据契约

## 本篇定位

第九篇补齐数据资产化、数据产品、价值评估和企业内部共享治理，使前文的数据流水线进一步沉淀为可发现、可复用、可审计的组织级资产。

## 术语口径

本篇统一使用“数据资产”描述具有来源、质量、权限、价值和生命周期记录的数据对象；使用“数据产品”描述具备用户、服务边界、SLA 和交付责任的数据资产形态；使用“数据契约”描述生产方与消费方之间关于 schema、质量、权限和变更的约定。资产目录、数据产品、数据市场和共享治理应分别说明治理对象，不混同为简单的数据列表。
