- 先读第31章，明确数据工程 Agent 的架构、任务边界和自动化等级。
- 再读第32章到第34章，理解采集清洗、标注合成评测和 DataOps 自治的核心闭环。
- 最后读第35章，把权限、安全、审计和人机协同作为上线前门禁。
