- 先读第29章和第30章，建立开源预训练与后训练的最新配方坐标。
- 再读第31章和第32章，理解推理模型与 VLM 数据工程的关键分化点。
- 最后读第33章及四个项目页，把多模态生成与复现实战串成落地路径。
