# 第十一篇：隐私合规与数据安全

## 本篇定位

第十一篇聚焦数据合规、隐私保护、联邦学习与安全边界，强调在工程流程中把法规要求、风险控制与可审计能力前置到数据生命周期。

## 术语口径

本篇统一使用“数据合规”描述用途、授权、留存、删除、跨境和审计等约束；使用“隐私保护技术”描述差分隐私、联邦学习、加密计算和最小化采集等工程手段；使用“安全边界”描述权限、隔离、访问控制和误用防护。合规要求不应写成附录式声明，而应进入数据生命周期、模型训练和上线运营的前置门禁。
